# 项目定位与设计原则

## 一、网站定位

该网站应定位为：

**学术风格个人主页（Academic Personal Website） + 交互式项目作品集（Interactive Research Portfolio）**

目标用户包括：

- 导师 / 合作者
- 招聘方 / 面试官
- 学术同行
- 竞赛评委或项目审阅者
- 对个人研究方向感兴趣的访问者

## 二、核心目标

网站需要完成以下目标：

1. 展示个人身份与研究方向
2. 系统展示科研项目、论文与软件成果
3. 展示竞赛、奖项与技术能力
4. 提供清晰的联系方式
5. 形成稳定、可信、专业的线上个人品牌
6. 为后续绑定自定义域名做好准备

## 三、设计原则

### 1. 学术风格优先
整体视觉应克制、简洁、可信，避免过于花哨或娱乐化。

### 2. 项目表达优先
科研项目、论文、竞赛、软件成果要比普通个人介绍更突出。

### 3. 交互增强但不过度
应加入适量交互，让网页更现代，但不能破坏专业感。

### 4. 后续可扩展
必须支持后续扩展：
- 博客
- 出版物页面
- 项目详情页
- 下载页面
- 域名绑定
- 多语言

## 四、整体风格关键词

- Academic
- Minimal
- Research-oriented
- Interactive
- Clean
- Elegant
- Trustworthy

## 五、不建议采用的风格

以下方向应避免：

- 过于商业化的企业宣传风
- 赛博朋克或高噪声视觉
- 大面积炫技型 3D 动效
- 复杂到影响加载速度的粒子背景
- 过度卡通化设计
