# 交互设计说明

## 一、交互目标

交互应服务于三个目的：

1. 提升浏览体验
2. 强化现代感与完成度
3. 不破坏学术风格

## 二、建议加入的交互

### 1. 深浅色模式切换
必须保留，属于高性价比功能。

### 2. 平滑滚动导航
点击导航自动滚动到对应区域。

### 3. 进入视口动画
模块在滚动进入时淡入上移，增强层次感。

### 4. 项目卡片 hover
推荐：
- 阴影增强
- 边框高亮
- 图片轻微缩放
- 箭头位移

### 5. 复制邮箱按钮
增强联系模块的可用性。

### 6. 筛选与搜索
在 Projects 页面中加入：
- Tag filter
- Search input

### 7. 详情页图片灯箱
项目图片点击放大浏览。

## 三、不建议过度加入的交互

- 全屏粒子动画
- 复杂 3D 场景
- 高频闪烁效果
- 过量视差滚动
- 自动播放音视频

## 四、推荐动画库

- Framer Motion

## 五、动效节奏建议

学术风格网站应偏“慢而稳”，不要采用过于活跃的弹跳或夸张动画。
