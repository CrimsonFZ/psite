# 后续扩展方向

## 一、可扩展模块

### 1. Publications 页面
展示论文、投稿状态、预印本、软件著作权等。

### 2. Blog 页面
用于记录研究笔记、竞赛总结、项目总结。

### 3. News / Updates
展示近期成果、会议、投稿、竞赛进展。

### 4. Gallery
展示项目截图、论文图表、系统界面等。

### 5. Multi-language
支持中英双语切换。

## 二、功能扩展

- 全站搜索
- 标签归档
- 论文引用格式复制
- 项目图集轮播
- 博客 Markdown 渲染
- RSS
- Analytics
