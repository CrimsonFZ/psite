# 组件说明

## 一、基础布局组件

### Navbar
功能：
- 顶部导航
- 桌面端菜单
- 移动端折叠菜单
- 当前页面高亮
- 主题切换按钮

### Footer
功能：
- 版权说明
- GitHub 链接
- Email 链接
- 简短签名

---

## 二、首页核心组件

### Hero
内容：
- 姓名
- 学校与专业
- 一句话研究定位
- 主要按钮（Projects / CV / Contact）

建议交互：
- 文本渐入
- 按钮轻微 hover 效果

### About
内容：
- 个人介绍
- 当前研究方向
- 关注的问题领域

### ResearchInterests
内容：
- 以标签或卡片展示研究兴趣

### Education
内容：
- 学校、专业、成绩、时间
- 可做时间线形式

### SelectedProjects
内容：
- 精选项目卡片
- 跳转详情页

### Awards
内容：
- 奖项时间线或徽章墙

### Skills
内容：
- 分类标签
- 可视化熟练领域

### ContactPreview
内容：
- 邮箱
- GitHub
- 联系按钮

---

## 三、项目页组件

### ProjectCard
字段：
- title
- period
- short description
- tags
- image
- link

### ProjectHeader
字段：
- title
- subtitle
- tags
- quick links

### ProjectTimeline
用于展示：
- Background
- Core Work
- Methods
- Outcomes

### ImageGallery
用于展示：
- 论文图
- 系统界面
- 项目截图

---

## 四、通用 UI 组件

### SectionTitle
统一章节标题样式

### Badge / TechTag
统一标签外观

### Button
统一按钮交互与层级
