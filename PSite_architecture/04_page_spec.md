# 页面级详细说明

## 1. Home 首页

### 目标
建立专业、清晰、可信的第一印象。

### 模块
- Hero
- About
- Research Interests
- Education
- Selected Projects
- Awards
- Skills
- Contact Preview

### 交互建议
- 首屏渐入动画
- 项目卡片 hover 动效
- 滚动进入动画
- 深浅色主题切换
- 平滑滚动导航

---

## 2. Projects 页面

### 目标
集中展示所有重点项目。

### 展示方式
- 项目卡片网格
- 支持按类型筛选
- 支持按关键词搜索（可选）
- 每张卡片点击进入详情页

### 推荐筛选维度
- Forecasting
- Simulation
- Web Platform
- AI / NLP
- Power Systems

---

## 3. 单项目详情页

### 每个项目详情页包含
- 项目标题
- 时间
- 项目背景
- 核心工作
- 方法实现
- 主要成果
- 技术栈
- 项目图示
- 相关链接（GitHub / PDF / Paper / Demo）

### 页面目标
使访问者能快速理解该项目的：
- 问题背景
- 方法路径
- 个人贡献
- 成果与价值

---

## 4. CV 页面

### 目标
提供正式简历阅读与下载入口。

### 推荐形式
- 页面中展示简历摘要
- 提供 Download PDF 按钮
- 若设计精力足够，可做网页版 CV 样式

---

## 5. Contact 页面

### 建议内容
- 邮箱
- GitHub
- 学校
- 简短说明（欢迎学术交流 / 合作联系）

### 可选功能
- 联系卡片
- 复制邮箱按钮
- 外链按钮
