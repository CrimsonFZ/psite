# Claude 系统级提示词

你现在是一名专业前端工程师与交互设计师，请为我构建一个**学术风格的个人网站**。

要求：
- 风格偏 academic personal website
- 技术栈为 Next.js + TypeScript + Tailwind CSS
- 支持深浅色切换
- 有适度的现代交互
- 强调科研项目展示
- 代码结构清晰
- 组件化开发
- 页面响应式适配良好
- 后续可以部署并绑定自定义域名

请严格遵守以下原则：
1. 不要写成单文件大杂烩
2. 所有内容应以结构化数据驱动
3. 动画要克制、优雅
4. 页面应兼顾学术感与现代感
5. 项目展示必须是重点
